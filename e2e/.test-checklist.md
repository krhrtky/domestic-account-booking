# E2E Test Execution Checklist

## Pre-Test Setup
- [ ] Playwright browsers installed: `npx playwright install chromium`
- [ ] `.env.local` configured with Supabase credentials
- [ ] `SUPABASE_SERVICE_ROLE_KEY` set in environment
- [ ] Port 3000 is available
- [ ] Supabase project is accessible

## Run Tests
- [ ] All tests pass: `npm run test:e2e`
- [ ] No test data left in database
- [ ] HTML report generated

## Post-Test Verification
- [ ] Review test report: `npx playwright show-report`
- [ ] Check for flaky tests (run multiple times)
- [ ] Verify cleanup (check Supabase Auth users)

## CI/CD Setup
- [ ] GitHub Secrets configured:
  - [ ] `NEXT_PUBLIC_SUPABASE_URL`
  - [ ] `NEXT_PUBLIC_SUPABASE_ANON_KEY`
  - [ ] `SUPABASE_SERVICE_ROLE_KEY`
- [ ] Workflow runs successfully on push
- [ ] Playwright report artifact uploaded

## Test Coverage Verification
- [ ] Login with valid credentials
- [ ] Login with invalid credentials
- [ ] Signup new user
- [ ] Signup validation (email, password)
- [ ] Dashboard access after login
- [ ] Dashboard shows settlement data
- [ ] Navigation links work
- [ ] Unauthenticated redirect
