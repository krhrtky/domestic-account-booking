# UI Component Rules

This rule applies to all React component files (`**/*.tsx`).

## Applicable Laws

Reference: [docs/laws/01-customer-experience.md](../../docs/laws/01-customer-experience.md)

### L-CX: Customer Experience

#### L-CX-002: UI Display Consistency
| Item | Format | Example |
|------|--------|---------|
| Currency | `¥{comma-separated}` | ¥10,000 |
| Date | `YYYY年MM月DD日` or `YYYY/MM/DD` | 2025年1月15日 |
| Percentage | `{number}%` | 60% |

Forbidden:
- Different formats across screens
- Currency without ¥ symbol
- Non-localized date formats (`Jan 15, 2025`)

#### L-CX-003: Error Message Clarity
- Be specific about what went wrong
- Provide actionable next steps
- Use non-technical Japanese language

Forbidden:
- Raw technical errors (`TypeError: Cannot read property...`)
- Vague messages (`エラーが発生しました`)
- English-only messages

#### L-CX-004: Feedback Immediacy
| Action | Max Time | Feedback |
|--------|----------|----------|
| Button click | 100ms | Loading state |
| Form submit | 200ms | Submitting indicator |
| API wait | 3s | Progress display |
| Timeout | 3s+ | Timeout message |

### L-LC-004: Prohibited Expressions

Never use these in UI text:

**A. Superiority claims:**
- 完璧, 100%, 絶対, 必ず, 確実
- 業界No.1, 最高, 最強, 究極, 唯一

**B. Professional advice:**
- 節税, 確定申告, 税金が減る
- 投資判断, 資産運用, 利益が出る
- 法的に有効, 契約として成立

**C. Discrimination:**
- 主婦向け, 男性の稼ぎ, 女性の家事
- 貧乏, 金持ち, 低所得
- 頭が悪い, 馬鹿でもわかる

**D. Fear appeals:**
- 損する, 危険, 今すぐやらないと
- 今だけ, 期間限定, 急いで
- 後悔, もったいない

### Allowed Messages
```typescript
'参考値としてご利用ください'
'正確な金額は各明細をご確認ください'
'税務・法務については専門家にご相談ください'
```
